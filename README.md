# MOA MTS4 Wireframe Prototype

이 폴더는 MOA MTS4 와이어프레임을 HTML 프로토타입으로 관리하는 공간입니다.

## 시작 파일

- `index.html`: 발표와 공유용 시작 페이지
- `moa_mts4_home_wireframe.html`: 실제 앱 홈 화면
- `moa_mts4_ai_fullscreen_wireframe.html`: MOA AI 메인 화면
- `moa_mts4_cascade_main.html`: 캐스케이드 메인 화면
- `moa_mts4_cascade_setup_edit_all.html`: 캐스케이드 빠른 설정 화면
- `moa_mts4_ai_strategy_studio.html`: AI 자동매매 운영 대시보드
- `moa_mts4_market.html`: 마켓 메인 화면

## 공통 파일

- `moa_mts4_common.js`: 공통 푸터, 전체 메뉴, 역할/흐름 보기
- `moa_mts4_cascade_quick.css`: 캐스케이드 설정 화면 스타일
- `moa_mts4_stock_trade.css`: 일반 주문 화면 스타일
- `moa_mts4_feature_flow.css`: 기능 플로우 화면 스타일

## 정리 기준

- 루트에는 실제로 열어보는 HTML/CSS/JS 파일만 둡니다.
- 검수 캡처, 발표용 PNG, 기능 명세 PDF는 `_archive` 폴더에 보관합니다.
- `_archive` 안의 파일은 화면 실행에는 필요하지 않은 참고 자료입니다.

## 온라인 공유

GitHub Pages, Netlify, Vercel 같은 정적 호스팅에 올릴 때는 이 `wireframes` 폴더를 사이트 루트로 사용하면 됩니다.
